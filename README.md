# Аналитические дашборды в Yandex DataLens

## Инструменты
- Yandex DataLens
- ClickHouse
- Excel / CSV

## Описание проектов

### ДТП на дорогах России
https://datalens.ru/4150bldn5tcoo-dashbord-dtp?state=13ed3945206

Дашборд на основе данных о ДТП за 2018 год. Включает карту плотности ДТП, графики смертности по регионам, а также временную динамику по неделям, дням недели и часам суток. Фильтры по региону, дороге, типу ДТП и типу дороги.

### Титаник
https://datalens.ru/77zzqytqyjeir-analiz-vyzhivaemosti-passazhirov-titanika

Анализ выживаемости пассажиров. Сводная таблица показывает количество пассажиров по классу обслуживания и возрастным группам, распределение выживших и погибших.

### Качество воздуха 
https://datalens.ru/dd690zxmsghix-vozduh

Дашборд на основе данных датчиков качества воздуха. Карта с двумя слоями (полигоны PM2.5 и точки PM10) показывает загрязнение. Графики отображают очаги и всплески ухудшения воздуха в Москве за последние 90 дней.

### Рестораны Москвы 
https://datalens.ru/lo5w9kl6dw9a5-analiz-restoranov-moskvy

Дашборд визуализирует заведения общепита Москвы. Карта показывает расположение ресторанов, кафе и баров. Диаграмма распределяет сетевые и несетевые заведения по административным округам.

### Анализ продаж сети магазинов 
https://datalens.ru/05cm2h48kczwk-analiz-prodazh-magazinov

Дашборд на основе данных о продажах сети магазинов Москвы. Включает динамику продаж по месяцам с разбивкой по типам оплаты, столбчатую диаграмму продаж по брендам и категориям товаров, сводную таблицу по годам и категориям, а также тепловую карту плотности заказов.

### Анализ объявлений auto.ru
https://datalens.ru/9elcfewb7vtet-avto-ru

Дашборд на основе данных о продаже автомобилей. Включает средний возраст авто по маркам, график изменения стоимости от возраста, распределение популярных цветов, зависимость цены от пробега.

### Анализ таксопарка
https://datalens.ru/diqc69w5sno2x-analiz-taksoparka

Дашборд на основе данных о поездках таксопарка за неделю. Включает карту с оценками поездок, ключевые метрики прибыли и расходов, график утилизации автомобилей, прибыль по водителям и средний чек по статусам клиентов.
